Penggunaan Tag TABLE pada HTML

Tag `< table >` pada HTML digunakan untuk membuat tabel yang terstruktur secara visual, yang mengorganisir data dalam baris dan kolom. Tabel HTML dapat berisi berbagai jenis data, mulai dari teks biasa hingga gambar dan bahkan elemen-elemen HTML lainnya. Setiap tabel terdiri dari satu atau lebih baris, yang masing-masing dapat berisi satu atau lebih sel. Tag `< table >` dapat diperluas dengan menggunakan tag tambahan seperti `< thead >`, `< tbody >`, dan `< tfoot >` untuk membagi tabel menjadi bagian-bagian yang lebih terstruktur, serta menggunakan tag seperti `< tr >` untuk baris dan `< td >` atau `< th >` untuk sel. Ini membuat tabel HTML menjadi alat yang fleksibel dan kuat untuk menyajikan informasi secara terstruktur di dalam halaman web.
